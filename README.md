## To open the file
1. Ensure that Python is installed on your device.
2. Download the app_p3.py file and save it to a local folder.
3. Open Command Prompt and navigate to the folder containing the file using the cd command.
4. Run the following command:
   python -m streamlit run app_p3.py
5. Once the application starts, open the URL displayed in the terminal (typically http://localhost:8501) in your web browser.
